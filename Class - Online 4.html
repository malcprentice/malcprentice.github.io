Aim of Online Unit 4 (Class 3)

In this class, you will learn how to:

  • make a participation plan
  • Continue to write an essay about ethical consumption
    • review paragraph basics
    • learn to use a template or format written work
    • learn to use title case
  • Review common issues and questions so far (Vocab, QW, Essay)
    • Some reminders about how to format written work
    • Some reminders about how to paragraph basics
  • Learn the last regular homework "SR": do a speed reading
  • Learn how to make a participation plan
