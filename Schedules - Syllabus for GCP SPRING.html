General Description

This class, which is only open to students in the Global Citizenship Program, is designed to help students prepare for fully participating in western-style English-only classes in their academic field.

Goals and Objectives

Objectives are same for both semesters but more is expected in semester two.

This course follows the Curriculum for General Academic Skills

Required Materials

A4 40 pocket plastic folder

Grading system

See grading system

Optional ways to get OK points

  • I want you to try what is available in SAC and the university.
  • One point each for TRYING the things below.

do well in class
see get independent learning points

Ways to get NG points

Be absent / late NG points only -1 point for absent, -0.5 for late no max. Also, 7 = fail! See do well in class
Forget homework/textbook NG points only -1 per incident no max See do well in class
Not paying attention NG points only -1 per incident no max See do well in class

Notes on assessment

  • Required items with OK/NG points: If you do LESS than the minimum, you get NG points up to the value of the max OK points and/or assigned points.
  • "Finished vocab"? Really? Are you sure? Even the new lists? OK, no minus points for not submitting 280 words. Ignore the warnings on your reports. However, I will test you on the full NewGSL+AWL lists at the end of term.

Projects Overview

Essay Topic Style
Spring A write an essay about ethical consumption 5 paragraph problem-solution essay
Spring B write an essay about endangered animals 5 paragraph problem-solution essay
Spring C write an essay about voluntourism criteria-analysis poster presentation
Fall A write an essay about refugees and asylum seekers 9 paragraph problem-solution essay
Fall B write a country case study 9 paragraph problem-solution essay
Fall C do a survey case study powerpoint presentation

  • At least 50% of all sources should be English. Academic sources are Journals or Books.
  • There is no word limit as such, but you must have:
    • No more or less than the required sentences-per-paragraph (see template)
    • No short sentences
