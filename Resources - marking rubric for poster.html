Grading your poster

I will grade your slides first, then I will ask you "Can you tell me a little more about problem/solution 1/2/3?"and grade your answer

__Delivery for Slides G
Font meaning consistent (size/colour/type) OK
Text alignment consistent OK
Bullet capitalisation consistent OK
use title case consistently? OK
Figure or table on most slides? OK
Too much enpty space? OK
Note English (key words not sentences) OK

__Arrangement for Slides G
* Correct sections/slides included OK
* Slide order clear? OK
* Enough Topics/Points/Details given? OK

__Style for Slides G
* Clear and easy to understand? OK

__Invention for Slides G
* Any plagiarism / bad paraphrasing? OK
* use APA format in the text OK
* use APA format in reference lists OK
* Enough sources used (see project sheet for number) OK
* Find good sources (worksheet) OK

__Invention/Arrangement/Style for Answers__ G
Can give a clear and organised answer to the question? OK
Give information not on slide OK

__Delivery for Answers__ G
* Interesting voice? OK
* Eye contact with humans (not slide/notes/floor) OK
* Gestures controlled or relaxed (no freeze/twitch/fists)OK
* Body - lean/sway/rock? OK
* Notes - fiddle/wave/drop? OK
* Any other signs of nervousness? OK
