# Quotes
Here is a list of discussion starting quotes, chosen using this random quote generator: http://www.quotationspage.com/random.php
You can discuss using these or other questions 
1) What do you think it means? 
2) Do you think this quote is true? 
3) Can you think of an example?


* The dead might as well try to speak to the living as the old to the young. (Willa Cather)
* The time you enjoy wasting is not wasted time. (Bertrand Russell)
* Music is essentially useless, as life is. (George Santayana)
* Be kind, for everyone you meet is fighting a hard battle. (Plato)
